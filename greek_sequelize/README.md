
# GreekRush v0.0.1

## Overview

This is the Beta build of GreekRush. GreekRush utilizes [AngularJS](https://docs.angularjs.org/api) and
[Firebase](https://www.firebase.com/docs/). In additio, the 
[AngularFire API](https://www.firebase.com/docs/angular/reference.html) is used to communicate between the 
front and back-end

## Start Server

To begin running the node server: 
```
$	cd ./greekrush.fore/greek_fire
$	grunt serve
```
Preview>Port:3000


## Code Standards and Naming Conventions

### Formatting
0. Tab settings should be set to regular 4 space (no soft tabs)
0. Ruler should be set to 120 characters. Use this as your max line length guide
0. Do not leave and blank space at the end of a page

###Controllers
 + path: / app / scripts / FOCUS / controllers / FUNCTION.js  [all lowercase if one word, otherwise twoWords ]
 + name: NameCtrl
 + module: greekRush.focus.controllers.funtion
